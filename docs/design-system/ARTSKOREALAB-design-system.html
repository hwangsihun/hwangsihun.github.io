Audit Notes

문서 UI
Pretendard를 우선 사용하고, 미설치 환경에서는 Noto Sans KR로 fallback됩니다.
토큰 구조
global utility와 layout component 규칙이 함께 디자인 시스템을 형성하므로 둘을 분리해서 읽는 것이 중요합니다.
재생성 가능성
HTML 원본과 PDF export 스크립트를 같이 제공해 이후 업데이트도 같은 방식으로 생성 가능하게 구성했습니다.
